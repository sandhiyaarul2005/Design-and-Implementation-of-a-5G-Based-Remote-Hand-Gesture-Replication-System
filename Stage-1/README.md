# 📡 Stage–1: Study and Performance Evaluation of 5G Standalone Network

## Overview

This stage focuses on the analytical study and performance evaluation of the 5G Standalone (SA) network architecture. The objective was to examine the feasibility of leveraging 5G SA infrastructure for low-latency, real-time gesture transmission and remote actuation systems.

The study emphasizes understanding the architectural framework of 5G SA and assessing its suitability for time-sensitive teleoperation applications.

---

## Technical Study

A structured analysis of the 5G Standalone architecture was conducted, covering:

- gNB (Next Generation NodeB)
- 5G Core (5GC)
- Access and Mobility Management Function (AMF)
- User Plane Function (UPF)
- Service-Based Architecture (SBA)

The architectural differences between Non-Standalone (NSA) and Standalone (SA) deployments were examined, highlighting the advantages of a fully integrated 5G core network in achieving reduced latency and enhanced reliability.

---

## Performance Evaluation

Empirical measurements were carried out to evaluate key network performance parameters:

- **Uplink Data Rate**
- **Downlink Data Rate**
- **End-to-End Latency (Round Trip Time)**
- **Jitter**
- **Packet Loss**
- **Signal Quality Indicators (RSRP, SINR)**

These parameters were analyzed to determine throughput consistency, network stability, and responsiveness under real-time communication scenarios.

---

## Observations and Inference

The performance evaluation indicates that the 5G Standalone network demonstrates:

- High bidirectional data throughput
- Reduced end-to-end latency
- Improved communication reliability
- Stable network behavior under continuous transmission

The observed characteristics validate the suitability of 5G SA for latency-critical applications such as remote gesture replication, teleoperation systems, and distributed robotic control frameworks.

---

## Relevance to Project

The findings from this stage establish the communication foundation required for implementing a 5G-enabled real-time gesture transmission system in subsequent stages of the project.
