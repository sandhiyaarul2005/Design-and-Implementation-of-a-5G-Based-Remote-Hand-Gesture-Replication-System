# Stage–3: Flex Sensor to Servo Angle Mapping

## Overview

Stage–3 focuses on implementing a real-time gesture-based actuation system by interfacing a flex sensor with a microcontroller and mapping its analog readings to the angular displacement of a servo motor.

This stage validates the direct relationship between sensor bending and proportional mechanical movement, forming the foundational sensing-to-actuation mechanism for the project.

---

## Objective

- To interface a flex sensor with the ESP32 microcontroller.
- To acquire real-time analog readings using the internal ADC.
- To calibrate minimum and maximum bending values.
- To map sensor readings into a servo motor angle range (0°–180°).
- To achieve smooth and proportional servo actuation.

---

## System Description

### 1. Flex Sensor Interfacing

The flex sensor operates on the principle of resistance variation with bending.  
It was connected in a voltage divider configuration, and the output voltage was provided to an analog input pin of the ESP32.

The ESP32’s ADC converts this analog voltage into corresponding digital values.

---

### 2. Sensor Calibration

To ensure accurate mapping:

- The minimum ADC value was recorded when the sensor was straight.
- The maximum ADC value was recorded at maximum bend.
- These limits were used to define the mapping range.

Calibration improves accuracy and prevents improper servo rotation.

---

### 3. Angle Mapping Logic

The ADC values were linearly mapped to the servo motor angle range using:

Mapped_Angle = map(ADC_Value, ADC_Min, ADC_Max, 0, 180);

Where:

- ADC_Min → Value at no bend  
- ADC_Max → Value at maximum bend  
- 0–180 → Servo angular displacement range  

This ensured proportional mechanical motion corresponding to finger bending.

---

### 4. Servo Motor Control

- An SG90 servo motor was controlled using PWM signals generated by the ESP32.
- The mapped angle value was written to the servo.
- Real-time rotation was achieved based on flex sensor bending.
- An external 5V supply was used to ensure stable servo operation.

---

## Hardware Components

- Flex Sensor (Resistive Bend Sensor)
- ESP32 Microcontroller
- SG90 Servo Motor
- External 5V Power Supply
- Connecting wires and voltage divider resistor

---

## Key Observations

- Servo angle varied proportionally with the degree of bending.
- Proper calibration significantly improved response accuracy.
- Stable power supply minimized jitter and unwanted resets.
- Linear mapping provided smooth and predictable actuation.

---

## Conclusion

Stage–3 successfully demonstrates real-time gesture-to-motion conversion using local sensing and actuation. The system establishes a reliable foundation for converting human motion into controlled mechanical output, which can be extended to advanced control systems in future stages.
